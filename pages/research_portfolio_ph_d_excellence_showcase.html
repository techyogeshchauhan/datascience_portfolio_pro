PhD Dissertation

AI-Based Model to Predict Depression in Women Using Multimodal Data Analysis

My doctoral research focuses on developing enhanced multimodal deep learning approaches to identify key factors influencing depression in women using physiological and behavioral data. This work combines machine learning with healthcare analytics to improve early detection and intervention strategies.

Multimodal Data Integration

Combining physiological signals, behavioral patterns, and self-reported data for comprehensive analysis

Deep Learning Architecture

Developing novel neural network architectures optimized for mental health prediction tasks

Clinical Validation

Rigorous testing and validation with clinical partners to ensure real-world applicability
